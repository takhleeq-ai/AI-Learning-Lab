# AI Learning Lab

A public, experiment-driven journey into machine learning and applied AI.

This repository documents my process of learning AI by building small,
focused experiments and explaining the results.

## Background

My professional background is in fintech and payments infrastructure
(Apple Tap to Pay ecosystem and Lloyds Banking Group).

Most of my work has been in regulated systems where reliability,
evaluation and failure modes matter more than novelty.

This repository extends that systems thinking into machine learning.

## Learning approach

Rather than collecting tutorials, I am learning through small experiments:

- build a dataset
- train a model
- evaluate the results
- document what worked and what didn't

Each experiment lives in its own folder.
